##  Project Description

This project implements three classical sorting algorithms:

* Insertion Sort
* Merge Sort
* Quick Sort

The goal is to analyze and compare their execution time on different input sizes using empirical runtime analysis.

---

## Algorithms Implemented

###  Insertion Sort

A simple sorting algorithm that builds the final sorted array one element at a time.

###  Merge Sort

A divide-and-conquer algorithm that splits the array and merges sorted subarrays.

###  Quick Sort

A fast sorting algorithm that uses a pivot to partition the array.
